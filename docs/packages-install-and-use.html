4.5.1 CRAN is the canonical R package resource

4.5.2 Using packages without library()-ing them

Some packages you don‚Äôt really want to have to go through the trouble of loading, you‚Äôll just want one function. For example, this website is generated with one function, render_book() in the package bookdown. Instead of:

render_book()

I can save a line by just using ::

This means I am looking in library bookdown for function render_book. The function is hidden again right afterwards, but it is useful if you just want to run one function from a library, once.
